## 概述

本项目遵循 **PEP 8** Python 命名规范，并根据项目架构特点制定以下补充规范。

**核心原则：**
- ✅ 清晰表达意图和职责
- ✅ 避免缩写，优先使用完整单词
- ✅ 保持命名的一致性
- ✅ 易于搜索和重构

## 类命名规范

### 1. 管理类 - `*Manager`
**适用场景：** 协调、管理多个实例或资源的类

```python
# ✅ 正确示例
class ScreenshotManager:
    """全局截图管理器，实现截图资源共享"""
    def get_full_screenshot(self, priority: int = 0):
        pass

class ImageDetectionManager:
    """管理多个图像检测任务的协调器"""
    image_detections: dict[int, ImageDetection]
    
    def start_all_detection(self):
        pass

class BackgroundMonitorManager:
    """管理多个后台监控任务"""
    monitors: dict[int, BackgroundMonitor]
    
    def start_all_groups(self):
        pass

class MemoryManager:
    """内存使用情况管理和垃圾回收"""
    def gc_if_needed(self, frame_count: int) -> bool:
        pass

# ❌ 错误示例
class ScreenshotUtil:        # 应该是 Manager
class ImageDetector:         # 不够清晰，改为 ImageDetectionManager
class MemoryMonitor:         # 应该用于监控，不是管理，改为 MemoryManager
```

---

### 2. 执行类 - `*Worker` 或 `*Task`
**适用场景：** 执行具体任务的类（通常被 Manager 类管理）

```python
# ✅ 正确示例
class ImageDetectionWorker:
    """执行单个图像检测任务"""
    group_index: int
    is_running: bool
    
    async def _detect_async(self):
        pass

class BackgroundMonitorWorker:
    """执行单个后台监控任务"""
    group_index: int
    hwnd: int
    
    async def _monitor_async(self):
        pass

class OCRRecognitionTask:
    """执行单次OCR识别任务"""
    def perform_recognition(self) -> str:
        pass

# ❌ 错误示例
class ImageDetection:         # 不够明确，改为 ImageDetectionWorker
class BackgroundMonitor:      # 应该是 BackgroundMonitorWorker
class OCRModule:              # 不够具体，改为 OCRRecognitionTask 或 OCRWorker
```

---

### 3. 识别器/转换器 - `*Recognizer` / `*Converter`
**适用场景：** 提供无状态或低状态的识别/转换服务

```python
# ✅ 正确示例
class OCRRecognizer:
    """OCR文字识别工具（无状态）"""
    @staticmethod
    def get_text(image, language: str) -> str:
        pass

class ImageRecognizer:
    """图像模板匹配识别工具"""
    @staticmethod
    def match_template(image, template, threshold: float) -> tuple:
        pass

class ColorRecognizer:
    """颜色识别工具"""
    @staticmethod
    def match_color(image, target_color, tolerance: int) -> tuple:
        pass

class CoordinateConverter:
    """坐标系转换工具"""
    @staticmethod
    def screen_to_window(screen_pos, window_offset):
        pass

# ❌ 错误示例
class OCRUtil:               # 改为 OCRRecognizer
class ImageMatcher:          # 改为 ImageRecognizer
class ColorDetector:         # 改为 ColorRecognizer
```

---

### 4. 监控类 - `*Monitor`
**适用场景：** 仅在真正的"监控"职责时使用（观察状态变化）

```python
# ✅ 正确示例
class ResourceMonitor:
    """监控系统资源（内存、CPU等）的状态变化"""
    def check_memory_usage(self) -> float:
        pass
    
    def check_cpu_usage(self) -> float:
        pass

class PerformanceMonitor:
    """监控性能指标"""
    def record_execution_time(self, duration: float):
        pass

# ⚠️ 谨慎使用
class BackgroundMonitor:
    # 这个类主要不是"监控"，而是"执行"后台任务
    # 应该改为 BackgroundMonitorWorker
```

```python
# ✅ 正确示例
class AppState:

class ModuleState:
    """表示单个模块的状态"""

```

## 函数/方法命名规范

### 1. 通用动词前缀规范

| 前缀 | 用途 | 返回值 | 示例 |
|------|------|--------|------|
| `get_*` | 获取数据，通常有计算 | 任意 | `get_memory_usage()` |
| `fetch_*` | 从远程/文件获取 | 任意 | `fetch_config()` |
| `set_*` | 设置属性/配置 | None 或 bool | `set_target_window(hwnd)` |
| `init_*` | 初始化 | None | `init_backend()` |
| `start_*` | 启动任务/服务 | None | `start_monitoring()` |
| `stop_*` | 停止任务/服务 | None | `stop_monitoring()` |
| `execute_*` | 执行操作 | 任意 | `execute_keypress(key)` |
| `perform_*` | 执行复杂操作 | 任意 | `perform_ocr_recognition()` |
| `is_*` | 判断状态（bool） | bool | `is_running()` |
| `has_*` | 检查是否拥有 | bool | `has_enabled_group()` |
| `can_*` | 检查是否可以 | bool | `can_execute()` |
| `validate_*` | 验证 | bool 或 tuple | `validate_region_coordinates()` |
| `calculate_*` | 计算 | 计算结果 | `calculate_min_interval()` |
| `update_*` | 更新 | None | `update_config()` |
| `sync_*` | 同步 | None | `sync_panel_configs()` |
| `clear_*` | 清除 | None | `clear_cache()` |
| `reset_*` | 重置 | None | `reset_state()` |

### 2. 异步方法命名

```python
# ✅ 正确示例
# 异步方法使用 async 关键字，命名保持一致
async def _ocr_group_loop(self, group_index: int, group: dict):
    """异步OCR识别循环"""
    pass

async def _monitor_async(self) -> None:
    """异步监控主循环"""
    pass

# ❌ 错误示例
async def ocr_loop_async():      # 改为 _ocr_group_loop
async def monitor():              # 不够清晰，改为 _monitor_async
```

### 3. 私有方法命名

```python
# ✅ 正确示例
class OCRModule:
    def start_monitoring(self):
        """公开接口"""
        pass
    
    def _validate_ocr_group_input(self, group) -> tuple:
        """私有验证方法（单下划线）"""
        pass
    
    async def _ocr_group_loop(self, group_index: int):
        """私有协程方法"""
        pass

# ❌ 错误示例
def validate_ocr_group_input():    # 应该加 _ 前缀表示私有
def __private_method():            # 双下划线过度防护，除非真的需要
```

```python
self.mgr = app                     # 缩写，改为 self.app
self.sr_mgr = ScreenshotManager()  # 缩写过度，改为 self.screenshot_manager
self.dict1 = {}                    # 无意义，改为 self.last_recognition_times
```

```python
for i, group in enumerate(self.app.ocr_groups):

for module_id, config in modules.items():

for group in self.app.ocr_groups:

for g in self.app.ocr_groups:     # 过度简写，改为 group
for x, y in enumerate(items):     # x, y 不清晰，改为 i, group
```

## 检查清单

在提交代码前，请按照以下检查清单验证命名规范：

### 类命名检查

- [ ] **管理类** 使用 `*Manager` 后缀
  - 示例：`ConfigManager`, `ScreenshotManager`, `ImageDetectionManager`
  
- [ ] **执行类** 使用 `*Worker` 或 `*Task` 后缀
  - 示例：`ImageDetectionWorker`, `BackgroundMonitorWorker`
  
- [ ] **识别/转换类** 使用 `*Recognizer` 或 `*Converter` 后缀
  - 示例：`OCRRecognizer`, `ImageRecognizer`, `CoordinateConverter`
  
- [ ] **监控类** 仅在真正监控时使用 `*Monitor`
  - 示例：`ResourceMonitor`, `PerformanceMonitor`
  
- [ ] **状态/事件类** 使用 `*State` 或 `*Event`
  - 示例：`AppState`, `ModuleState`

### 函数/方法检查

- [ ] 方法名长度 3-50 字符（超长应重构）
- [ ] 使用动词前缀（get, set, start, stop 等）
- [ ] 私有方法使用单下划线 `_`
- [ ] 异步方法使用 `async` 关键字
- [ ] 参数名清晰，避免单字母（i, g, x 等除外）

### 变量检查

- [ ] 避免缩写，除非非常标准（如 i, j, k 用于循环）
- [ ] 类型明确，必要时使用类型注解
- [ ] 集合变量使用复数（groups, items, configs）
- [ ] 布尔变量使用 is_, has_, can_ 前缀

### 常量检查

- [ ] 全局常量使用全大写 + 下划线
- [ ] 类常量使用全大写 + 下划线
- [ ] 相关常量组织为常量类或模块

### 文件检查

- [ ] 模块文件名使用小写 + 下划线
- [ ] 避免文件名过长或过于通用
- [ ] 相关功能文件放在对应目录

## 常见错误修正对照表

| 错误命名 | 正确命名 | 原因 |
|---------|---------|------|
| `ImageDetection` | `ImageDetectionWorker` | 明确职责是执行单个任务 |
| `BackgroundMonitor` | `BackgroundMonitorWorker` | 主要职责是执行，不是监控 |
| `MemoryMonitor` | `MemoryManager` | 职责是管理内存，不是被动监控 |
| `ocr_mod` | `ocr_module` 或 `_OCR` | 避免缩写 |
| `get_screenshot` | `get_full_screenshot` | 明确说明是全屏截图 |
| `recognize` | `recognize_once` | 明确说明执行次数 |
| `validate` | `validate_region_coordinates` | 说明验证的对象 |
| `process_image` | `_preprocess_image` | 加下划线表示私有函数 |
| `Priority_Number` | `PRIORITY_NUMBER` | 常量应全大写 |
| `bgm` | `background_manager` | 避免缩写 |
