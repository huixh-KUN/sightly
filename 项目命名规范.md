# 项目命名规范

遵循 PEP 8，补充以下项目约定。

## 类命名后缀
| 后缀 | 场景 |
|------|------|
| `*Manager` | 协调管理多个实例/资源 |
| `*Worker` | 执行具体任务（被 Manager 管理） |
| `*Task` | 执行单次任务 |
| `*Recognizer` | 无状态识别/转换服务 |
| `*Converter` | 转换器 |
| `*Monitor` | 监控状态变化（仅真正监控时用） |
| `*State` | 表示状态 |
| `*Event` | 表示事件 |
| `*Config` | 配置 |
| `*Builder` | 构建器 |
| `*Handler` | 处理器，被动响应事件/操作 |
| `*Controller` | 控制器，主动控制流程协调交互 |
| `*Backend` | 策略模式后端实现（输入/存储等可替换后端） |
| `*Picker` | 选择器组件（从候选中挑选） |
| `*Selector` | 选择器（窗口/文件等选择触发器） |
| `*Switcher` | 切换器（主题/模式等切换） |
| `*Viewer` | 只读查看组件 |
| `*Notifier` | 通知器（发出状态/事件通知） |
| `*Context` | 上下文/依赖容器 |
| `*Proxy` | 代理（延迟加载/访问控制） |
| `*Panel` | UI 面板（顶层页面） |
| `*Widget` | UI 小组件 |
| `*Dialog` / `*Window` | 对话框/窗口 |
| `*Frame` | 框架组件 |
| `*Button` | 按钮组件 |
| `*Card` | 卡片组件（QFrame） |
| `*Overlay` | 全屏覆盖层 |

**Handler vs Controller：** Handler 被动响应、低状态；Controller 主动控制、可能含业务逻辑。

**已废弃后缀（归并）：** `*Executor` → `*Worker`；`*Module` → `*Worker`/`*Manager`（按是否含生命周期管理选择）。

## 函数/方法命名
| 前缀 | 返回值 | 说明 |
|------|--------|------|
| `get_*` | 任意 | 获取数据 |
| `set_*` | None/bool | 设置属性 |
| `init_*` | None | 初始化 |
| `start_*` / `stop_*` | None | 启动/停止 |
| `is_*` / `has_*` / `can_*` | bool | 判断 |
| `fetch_*` | 任意 | 从远程/文件获取 |
| `load_*` | 任意 | 从文件/存储加载到内存 |
| `save_*` | bool/None | 持久化到文件/存储 |
| `validate_*` | bool/tuple | 验证 |
| `update_*` / `sync_*` / `apply_*` | None | 更新/同步/应用 |
| `clear_*` / `reset_*` | None | 清除/重置 |
| `execute_*` / `perform_*` | 任意 | 执行 |
| `create_*` | 新建对象 | 创建并返回新实例/资源 |
| `delete_*` | None | 删除资源 |
| `register_*` | None | 注册到注册表/映射 |
| `bind_*` | None | 绑定信号/回调 |
| `request_*` | None | 发出请求信号（不直接执行） |
| `collect_*` | 任意 | 从多处收集聚合数据 |
| `detect_*` | 任意 | 检测并返回检测结果 |
| `test_*` | dict/任意 | 单次测试，返回结果详情 |
| `select_*` | 任意 | 选择并返回所选对象 |
| `switch_*` | None | 切换当前对象/模式 |

- 私有方法用单下划线 `_`，避免双下划线
- 异步方法加 `async`，命名保持一致

## 变量
- 避免缩写，优先完整单词（`controller` 不写 `app`/`ctrl`）
- 集合用复数（groups, configs）
- 布尔用 `is_` `has_` `can_` 前缀

## 常量
全局常量和类常量用全大写+下划线：`PRIORITY_NUMBER = 6`

## 文件命名
- 小写+下划线：`ocr.py`, `image.py`, `click_worker.py`
- 避免过长和冗余：`ocr_recognition.py` → `ocr.py`
- 跨目录避免同名：`utils/image.py` 与 `modules/image.py` 应改名区分（如 `utils/image_utils.py`）
- 文件名应与主类名一致：`ScreenCaptureOverlay` → `screen_capture_overlay.py`，非 `screenshot.py`
- 导入用 `from ... import ClassName`，避免缩写别名
